# O que é este projeto?

Este projeto é uma iniciativa de **Aprendizado de Máquina** voltada à previsão de variáveis climáticas críticas no **Distrito Federal**.

Ele integra dados meteorológicos, hidrológicos e ambientais para apoiar a tomada de decisão no agronegócio, especialmente em culturas como **soja** e **milho**.

---

## Por que isso importa?

A produção agrícola no Distrito Federal depende fortemente da sazonalidade climática da região.

O DF apresenta dois períodos bem definidos:

- **estação seca:** maio a setembro
- **estação chuvosa:** outubro a abril

Essa variação impacta diretamente a irrigação, o manejo hídrico, o planejamento da safra e o uso eficiente de recursos naturais.

Com previsões mais precisas, torna-se possível:

- reduzir perdas
- otimizar recursos hídricos
- melhorar o planejamento agrícola
- apoiar decisões com base em dados

---

## O que o projeto prevê?

O projeto busca construir uma base sólida para prever:

- **precipitação acumulada**
- **evapotranspiração**
- **balanço hídrico**

Essas variáveis são essenciais para o manejo agrícola eficiente e para o acompanhamento das condições climáticas no Cerrado do Distrito Federal.

---

## Como o projeto funciona?

A proposta integra múltiplas fontes públicas e confiáveis, combinando diferentes escalas de observação:

- **escala local** — estações meteorológicas e pluviométricas
- **escala regional** — dados hidrológicos e de bacias
- **escala global** — reanálises e bases climáticas amplas

Essa abordagem busca aumentar a robustez do projeto, reduzir viés e ampliar a cobertura espacial e temporal dos dados.
