## Uso de dados

Todas as fontes utilizadas são públicas ou abertas para uso acadêmico e científico.

## LGPD

O projeto não utiliza dados pessoais identificáveis. Os dados trabalhados são de natureza:

- meteorológica
- hidrológica
- geofísica

## Transparência e reprodutibilidade

O projeto busca manter:

- fontes documentadas
- processo reproduzível
- organização clara da documentação
